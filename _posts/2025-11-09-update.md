## Subprojects and Dependency Updates

* [**cluster-api-provider-vsphere v1.15.0-beta.0**](https://github.com/kubernetes-sigs/cluster-api-provider-vsphere/releases/tag/v1.15.0-beta.0) introduces the next beta version of CAPV for testing upcoming Cluster API v1.15 functionality on vSphere. This release is intended only for testing and feedback.
* [**vsphere-csi-driver v3.6.0**](https://github.com/kubernetes-sigs/vsphere-csi-driver/releases/tag/v3.6.0) adds compatibility with Kubernetes v1.34 and brings improvements such as shared session support on vCenter login and enhanced task monitoring. Updated manifests for this release are available under the versioned `manifests/vanilla` directory.
* [**kustomize kyaml v0.21.0**](https://github.com/kubernetes-sigs/kustomize/releases/tag/kyaml/v0.21.0) updates structured data replacement capabilities, upgrades Go to 1.24.6, refreshes dependencies following security alerts, and includes minor YAML handling fixes.
* [**kustomize v5.8.0**](https://github.com/kubernetes-sigs/kustomize/releases/tag/kustomize/v5.8.0) enhances YAML/JSON replacement features, fixes namespace propagation for Helm integrations, and adds improvements such as regex support for replacements, new patch argument types, validation fixes, improved error messages, and performance optimizations.
* [**kustomize cmd/config v0.21.0**](https://github.com/kubernetes-sigs/kustomize/releases/tag/cmd/config/v0.21.0) aligns with kyaml updates, adopts Go 1.24.6, and brings dependency updates based on recent security advisories.
* [**kustomize api v0.21.0**](https://github.com/kubernetes-sigs/kustomize/releases/tag/api/v0.21.0) includes structured-data replacement enhancements, regex selector support, patch argument additions, namespace propagation fixes, validation improvements, Go 1.24.6 updates, and dependency refreshes.
* [**etcd v3.6.6**](https://github.com/etcd-io/etcd/releases/tag/v3.6.6) provides a new patch update for the v3.6 series with all changes documented in the linked changelog. Installation steps and supported platform updates are also included.
* [**etcd v3.5.25**](https://github.com/etcd-io/etcd/releases/tag/v3.5.25) delivers maintenance updates for the v3.5 series along with relevant upgrade guidance and support documentation.
* [**etcd v3.4.39**](https://github.com/etcd-io/etcd/releases/tag/v3.4.39) introduces the newest patches for the v3.4 branch with installation instructions and detailed platform support notes.
* [**cri-o v1.34.2**](https://github.com/cri-o/cri-o/releases/tag/v1.34.2) improves GRPC debug log formatting and ships updated, signed release bundles and SPDX SBOMs for all supported architectures.
* [**cri-o v1.33.6**](https://github.com/cri-o/cri-o/releases/tag/v1.33.6) publishes refreshed signed artifacts and SPDX documents for the 1.33 line, with no dependency changes recorded.
* [**cri-o v1.32.10**](https://github.com/cri-o/cri-o/releases/tag/v1.32.10) updates the 1.32 branch with new signed release artifacts and SBOM files, without dependency modifications.
* [**nerdctl v2.2.0**](https://github.com/containerd/nerdctl/releases/tag/v2.2.0) fixes a namestore path issue, adds mount-manager support, introduces checkpoint lifecycle commands, and enhances image conversion through a new estargz helper flag. The full bundle includes updated containerd, runc, BuildKit, and Stargz Snapshotter.
