## Release Schedule

**Next Deadline: [Code & Test Freeze](https://github.com/kubernetes/sig-release/blob/master/releases/release_phases.md#code-freeze), 19th March 2026**

Code and Test Freeze for v1.36 is coming up. All feature work must be completed
and merged before the freeze. After this point, only critical bug fixes will be
considered, and approved exceptions will be required for any additional changes.

Out-of-band patch releases v1.35.2, v1.34.5, v1.33.9, and v1.32.13 were published
last week, built with Go 1.25.7 / 1.24.13 and including fixes for recent CVEs.

Additionally, Kubernetes [v1.36.0-alpha.2](https://github.com/kubernetes/kubernetes/releases/tag/v1.36.0-alpha.2) is now live and built with Go 1.25.7.
