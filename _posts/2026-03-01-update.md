## Featured PRs

* [#137173: Graduate statusz API to beta](https://github.com/kubernetes/kubernetes/pull/137173)
  
The `config.k8s.io/statusz` API has been promoted from `v1alpha1` to `v1beta1`. This formalizes the statusz endpoint for component diagnostics and introduces version negotiation support, including deprecation handling for the alpha version.

* [#137172: Graduate Mixed Version Proxy feature gate to Beta](https://github.com/kubernetes/kubernetes/pull/137172)

The `UnknownVersionInteroperabilityProxy` (Mixed Version Proxy) feature gate has graduated to Beta and is now enabled by default. This stabilizes proxy behavior for version-skewed clusters during upgrades, with `--peer-ca-file` required to activate the proxy.
