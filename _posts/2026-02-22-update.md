## Other Merges

* [DaemonSet controller defers](https://github.com/kubernetes/kubernetes/pull/134937) sync on stale cache to prevent duplicate pod creation; adds daemonset_controller_stale_sync_skips_total metric (override via StaleControllerConsistencyDaemonSet=false).
* [Return PreBind-preempted Pods to the backoff queue](https://github.com/kubernetes/kubernetes/pull/135502) instead of deleting them from the apiserver.
* Networkpolicy: migrate [ipBlock.cidr validation to DV](https://github.com/kubernetes/kubernetes/pull/135715)
* [Fix kubectl plugin list](https://github.com/kubernetes/kubernetes/pull/136689) to detect overshadowed plugins correctly on Windows.
* [Add PodGroup scheduling cycle](https://github.com/kubernetes/kubernetes/pull/136618) so kube-scheduler can place an entire PodGroup in a single cycle.
* [Fix kubectl logs -f](https://github.com/kubernetes/kubernetes/pull/136411) to wait for containers to start instead of failing during ContainerCreating/PodInitializing.
* [Updated feature gate `MutablePodResourcesForSuspendedJobs` and `MutableSchedulingDirectivesForSuspendedJobs`](https://github.com/kubernetes/kubernetes/pull/135965) to be enabled by default.
* [Extended PostFilterResult](https://github.com/kubernetes/kubernetes/pull/136254) with a list of Pods that the PostFilter plugin preempted.
* Fix incorrect APIVersion in [PodCertificateRequest OwnerReference](https://github.com/kubernetes/kubernetes/pull/137008)
* [Add kubectl diff --show-secret](https://github.com/kubernetes/kubernetes/pull/137019) to explicitly allow displaying Secret values during diffs.
* Add an [alpha informer_processing_latency_seconds histogram metric](https://github.com/kubernetes/kubernetes/pull/137101) to measure event handler execution time in RealFIFO.
* Drop [ProtoMessage() markers](https://github.com/kubernetes/kubernetes/pull/137084) from k8s.io/api REST types; use apimachinery protobuf serializer instead (KEP-5589).
* [apiserver_rerouted_request_total metric](https://github.com/kubernetes/kubernetes/pull/137063) will expose labels for group, version and resource.
