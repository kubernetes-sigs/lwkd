## Other Merges

* [kube-proxy does not perform full-sync operations](https://github.com/kubernetes/kubernetes/pull/138571) when operating in large cluster mode (more than 1000 endpoints)
* Fixed [kubelet failure starting on ZFS](https://github.com/kubernetes/kubernetes/pull/138587) due to missing cadvisor plugin
* kubeadm: during [`kubeadm init`](https://github.com/kubernetes/kubernetes/pull/138449), if the default `admin.conf` and `super-admin.conf` paths are used, load the files but construct in-memory kubeconfigs that point to the `InitConfiguration.localAPIEndpoint` instead of the `ClusterConfiguration.controlPlaneEndpoint`, resolving issues with delayed load balancers provisioned only after the first kube-apiserver instance starts
* [Introduce Deferred Gen concept](https://github.com/kubernetes/kubernetes/pull/138205) to the Validation-gen framework
* [Improved CEL error messages in Dynamic Resource Allocation](https://github.com/kubernetes/kubernetes/pull/136709) to provide guidance when accessing non-existent device attributes, with links to documentation on handling optional fields using `orValue()` and `has()`
