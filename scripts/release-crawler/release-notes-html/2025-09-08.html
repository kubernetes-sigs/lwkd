Compatible containerd versions

This release of nerdctl is expected to be used with containerd v1.7, v2.0, or v2.1. Some features may not work with other releases of containerd.

[!NOTE]

Since this release, nerdctl is no longer tested with containerd v1.6.

About the binaries

  • Minimal (nerdctl-2.1.4-linux-amd64.tar.gz): nerdctl only
  • Full (nerdctl-full-2.1.4-linux-amd64.tar.gz): Includes dependencies such as containerd, runc, and CNI

Quick start

Rootful

$ sudo systemctl enable --now containerd
$ sudo nerdctl run -d --name nginx -p 80:80 nginx:alpine

Rootless

$ containerd-rootless-setuptool.sh install
$ nerdctl run -d --name nginx -p 8080:80 nginx:alpine

Enabling cgroup v2 is highly recommended for rootless mode, see https://rootlesscontaine.rs/getting-started/common/cgroup2/ .

The binaries were built automatically on GitHub Actions. The build log is available for 90 days: https://github.com/containerd/nerdctl/actions/runs/17484930617
